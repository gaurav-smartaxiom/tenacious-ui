User Manual

The dashboard is a central hub that provides a comprehensive summary of various metrics related to your shipments and trackers. Here's a quick overview of what you can expect to find:

  • 1. IMPACT trackers:
    This widget displays the number of active and inactive trackers currently in use.
  • 2. Total Shipments:
    This widget summarizes the number of shipments in your organization, and also provides a breakdown of counts by shipment type.
  • 3. Alerts:
    This widget showcases alerts related to trackers' battery, temperature and humidity ranges, allowing you to take proactive steps to protect your shipments.
  • Shipments:
    This list provides a detailed view of all the shipments which have data in last 24 hours, including their current status. Clicking on "Show All"
    displays the last location of all your shipments, while clicking on the shipment name displays its route and location on the map.
    4. Filter: The filter option can be used to find or filter shipments based on Shipment Name.
    5. Shipment Data: A user can check any shipment's route and details by locationg the shipment name in this list.
    6. Show Route: The button facilitates user to check the route covered by a particular shipment.
    7. Show Deatils: The button helps user to go through the complete details of any shipment.
  • 8. Map:
    It provides a visual representation of the location and route of your shipments, helping you keep track of their progress in real-time.
  • 9. Event Log:
    This table shows all the event details related to your shipments, providing you with a detailed record of their journey.

Tracker management is a crucial aspect of monitoring and maintaining the performance of your tracking devices. Here is a summary of the key features and actions available in the tracker management module:

  • Discover Tracker:
    To discover a tracker, simply navigate to the "Tracker Management" section from the navigation bar. Here you will find a table that provides detailed information about each tracker, including its UUID, serial number, shipment name (if assigned), battery status, activation status, calibration certificate, and version information.
    1. Filter: The filters facilitate a user to filter trackers based on UUID, serail number, shipment name, battery level, status and last connected.
    2. Widgets: The widgets provide a complete overview of the counts of trackers based on connectivity status. It also includes a progress bar.
    3. Callibration Certificate: User can download Callibration certificate of any tracker from here.
    4. More Info: By clicking on this button, a user can fetch version details of each tracker.
Activate Tracker
To activate a tracker, go to the "Shipments" section from the navigation bar and Create a new Shipment.
You can then assign the tracker to the shipment to begin monitoring its data.
Update Tracker Configuration
If you need to update the configuration of a tracker, you can do so by updating the shipment associated with that tracker.
Simply navigate to the "Shipments" section and Update Shipment to reflect the new configuration.
Deactivate Tracker
To deactivate a tracker, navigate to the "Shipments" section, Update Shipment and unassign the tracker from the shipment. This will automatically deactivate the tracker.

User management is an essential aspect of maintaining control and access within your organization's system. Here is a summary of the key features and actions available in the user management module:

  • Create User
  • Update User
  • User Session History
  • Delete User
Steps to add or update shipment

To create a new user, simply click on the 'Add user' icon located on the top bar. This will redirect you to the 'Add New User' form where you can fill in the necessary details.
Once you have completed the form, click on 'Create Account' to register the new user in your organization.
If you wish to assign shipments to the user immediately, you can do so during this process. Alternatively, you can assign shipments to the user at a later time.

If you need to delete a user, simply navigate to the "User Management" page. From here, you can delete the user by clicking on the "Delete" icon located beside the user's data in the Actions column.
To edit any user's data, simply navigate to the "User Management" section under "Access Control". This will take you to a page that contains a table with a list of all users and their basic details.
From the Action column, click on the "Edit" icon to access the user's details. Here, you can edit various information about the user, including their username, address, and contact number.
User can also enable or disable email alerts and daily reports for the user. User can manage the user's role. If the assigned role is "User", you can assign or unassign shipments to the user.
Once you have made the necessary changes, click on "Update" to update the user's data.

From the navigation bar, select "Shipments", you will be directed to Shipments page.
The page consists of a table which gives a brief overview of the all the shipments. You can click on any of the shipment to view its complete report.

  • 1. Create Shipment:
    To create a new shipment, you have two options:
    1. From the Dashboard:
      • Click on the 'Add' icon located in the 'Total Shipments' widget on the dashboard.
      • This will take you to the shipment creation process.
    2. From the Navigation Bar:
      • Select 'Shipments' from the navigation bar.
      • Click on the 'Add New Shipment' button.
      • This will start the shipment creation process.

    The shipment creation process consists of four steps which are mentioned below.

  • 2. Refresh: By clicking on this button, user can fetch latest shipments' data. 3. Search: Search option facilitates user to search a shipment by its name, type or tracker id. 4. Delete Shipment: A user can delete a shipment using this icon.
  • 5. Update Shipment:
    1. Navigate to the "Shipments" page from the navigation bar.
    2. On the Shipments page, use the search options available at the top of the table to find the specific shipment you need to update.
    3. Once you've located the shipment, look for the corresponding 'Edit' icon in the Actions column and click on it.
    4. You will be presented with an update modal that consists of four sections similar to add shipment.
Steps to add or update shipments.
Shipment Details
In this step, you need to provide the basic information about the shipment, including:
  • Shipment name
  • Shipment type
  • Pickup and delivery details
  • Geofence (if required)
Gateway Mapping
In this step, you can assign a tracker to the shipment from the available options. To do this:
  1. Select a tracker from the list of available trackers.
  2. An accordion tab will open up where you can configure temperature, humidity settings, and the frequency of receiving events.
  3. Once you have filled in the details, click on 'Save' to send an activation request to the tracker.
Blutag Mapping
If you want to fetch data from a blutag, you can attach blutags from the list of available tags. The data from the blutags will be appended to the tracker data and pushed to the system.
Status
A shipment can have one of the following statuses:
  • Started: Indicates that the shipment is currently in transit, and data recording is active.
  • Paused: Indicates that the shipment is paused, and data recording is not required.
  • Completed: Indicates that the shipment has reached its destination, and data recording is no longer needed. The tracker will be unassigned and deactivated when the shipment is completed.
  • Cancelled: Indicates that the shipment has been cancelled, and data recording is not necessary.
To monitor a shipment, you can follow these steps:
  • On the Dashboard:
    • Click on the shipment's name from the 'Shipment List' on the left-hand side.
    • You can click on 'Show Details' to view all the specific details of the shipment.
    • The route of the shipment will be displayed on the map, giving you a visual representation of its progress.
  • Navigate to the Shipments page:
    • Locate and click on the specific shipment you wish to monitor.
    • You will be taken to the shipment's report page.
    • The report page provides comprehensive information about the shipment.
    • At the top, you will see the shipment's name, type, assigned tracker, tracker's battery percentage, signal strength, and the total recorded events.
    • Below this, you will find options to view the calibration certificate and download the report.
    • The page also displays the basic details and current data of the shipment.
    • A map is provided to show the complete route taken by the shipment.
    • Following the map, you will find alerts and a summary of the shipment.
    • Graphs for temperature and humidity are included, presenting the minimum, maximum, average, and mean values.
    • If any blutag is assigned, a separate graph for blutag temperature will be reported.
    • Finally, a table is presented that includes all the recorded events for the shipment.
Daily Report
The Superadmin, Admins and all users for whom daily reports have been enabled, will receive a report everday 9 A.M. which consists of all the events occured in last 24 hours.
Download/Email Report
Users can also Download/Email the report whenever they wish to. From the Top Bar, click on the PDF icon, from the popup choose Download/Email option.
